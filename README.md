# Analiza jakości zdjęć na podstawie histogramu

Program w języku Python analizuje jakość zdjęcia na podstawie histogramu jasności
oraz histogramów kanałów RGB.

Zakres działania programu:
- wczytanie zdalnego obrazu z wykorzystaniem linku URL
- wyświetlenie histogramu dla całego obrazu
- wyświetlenie histogramów dla kanałów R, G i B
- analiza jakości zdjęcia na podstawie wartości histogramu
- automatyczna ocena jakości obrazu (dobra / średnia / słaba)
- bonus: prosta poprawa jakości obrazu ocenionego jako słaby

Wykorzystane biblioteki:
- Python
- Pillow
- NumPy
- Matplotlib
